## Installation

just copy to your local agent skills directory (~/.codex/skills, ~/.claude/skills/, ~/.pi/agent/skills/ or ~/.agents/skills/ etc)

or install via npm (let's you pick what you want to install):

```bash
npx skills@latest add byteowlz/skillissues

```
